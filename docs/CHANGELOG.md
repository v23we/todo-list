# CHANGELOG

## 2026-03-26

- 修复“下一步”和子任务事实源重复的问题：有子任务时首页“下一步”改为自动取第一个未完成子任务
- 修复有子任务时首页和任务页错误直接完成父任务的问题，改为逐步完成子任务并在全部完成后自动结算父任务
- 补齐任务详情页子任务展开、逐个勾选、进度展示与自动完成父任务链路
- 明确“稍后”状态：不进入首页当前任务与备选补位，并在任务页中做弱化展示与恢复入口
- 修复奖励页 XP 样式切换无真实反馈的问题，新增 `小闪电 / 小液滴 / 小咖啡豆` 三种可见样式
- 为完成反馈补充轻量完成动效差异，并轻增强首页顶部等级卡片的层级与信息量

- 新增轻量 MVVM 与服务层，落地固定 `+20 XP`、升级、奖励解锁主链路
- 新增首页、任务页、奖励页、设置页与任务详情/创建页
- 实现首页 `1 个当前任务 + 2 个备选任务` 和当前任务补位规则
- 新增主题切换、XP 样式占位、音效/完成动效占位
- 新增基础单元测试与项目说明文档
